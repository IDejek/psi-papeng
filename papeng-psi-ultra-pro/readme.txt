Ultra-premium enterprise-grade WordPress theme and plugin for DPW PSI Papua Pegunungan.

== Description ==

Complete WordPress solution for DPW Partai Solidaritas Indonesia Papua Pegunungan.

**Theme Features:**
* Real-time WIT/WIB clock topbar
* Premium hero slider (admin-managed)
* Leadership profiles with modal popups
* Organizational divisions section
* News & articles with SEO schema
* DPD (regional branch) management
* Video gallery with YouTube integration
* Photo gallery with lightbox
* Contact form with AJAX
* Full SEO support (Schema.org, Open Graph, Twitter Cards, XML Sitemap)
* Responsive & mobile-first design
* Glassmorphism & premium animations
* Custom login page

**Plugin Features:**
* Member management system (CRUD)
* Member verification workflow
* Member statistics with charts
* Activity logging
* SMTP email configuration
* WhatsApp integration
* CSV export
* Contact form handler

== Installation ==

1. Upload `psi-papeng-theme` folder to `/wp-content/themes/`
2. Upload `papeng-psi-ultra-pro` folder to `/wp-content/plugins/`
3. Activate the theme and plugin via WordPress admin
4. Go to PSI Premium > Settings to configure
5. Go to Appearance > Menus to set up navigation
6. Go to Appearance > Customize for visual settings
7. Add content via the PSI Custom Post Types menus
